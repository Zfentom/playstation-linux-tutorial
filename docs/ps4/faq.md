# Frequently asked questions

## Q: Whats the latest kernel version for the ps4?
A: The lastest kernel version is 7.0.

## Q: How good is baikal support?
A: Baikal support has been greatly improved since the new developers joined the scene, here is the current status:

| Kernel version | Status |
|----------------|--------|
| 6.19 | ⚠️ Almost works |
| 5.15 | ✔️ (Not recommended for performance) |
| 7.0 | ⚠️ Soon to get ported |

## Q: Does baikal support internal install?
A: Yes baikal supports internal install now.

## Q: Is any other operating system supported?
A: Now as of now no other OS is supported but there are possible ones that can work like OpenBSD,FreeBSD and other such distros. **AND NO WINDOWS IS NOT AND NEVER WILL BE SUPPORTED**

## Q: Can you emulate games on it?
A: Yes you can emulate pretty much any game that is not ps4 of course becase shadps4 is not supported. Yes you can emulate ps3 games(Not at full speed for most of the library).
