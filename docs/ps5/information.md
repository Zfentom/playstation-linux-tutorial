# Key information
This tutorial will cover how to install Linux on your PS5. It is created to make it easier for people to install linux on their consoles.

>[!NOTE] Will this affect my games? can it reverse it?
> All your info should stay intact because you will install it on a seperate M.2 drive.

::: details Which consoles are compatible?
ps5-linux is currently only supported on PS5 Phat on following firmwares

| Firmware | Support | M.2 |
|----------|---------|-----|
| 6.02 | ✔️ | ✔️ |
| 4.51 | ✔️ | ✔️ |
| 4.50 | ✔️ | ✔️ |
| 4.03 | ✔️ | ✔️ |
| 4.02 | ✔️ | ✔️ |
| 4.00 | ✔️ | ✔️ |
| 3.21 | ✔️ | ❌ |
| 3.20 | ✔️ | ❌ |
| 3.10 | ✔️ | ❌ |
| 3.00 | ✔️ | ❌ |

- Support for 1.xx and 2.xx firmwares may be added in the future, but we will not prioritize this effort.

If you want to update to a specific firmware, [download the correct PUP](https://darthsternie.net/ps5-firmwares/) and follow the [official guide](https://www.playstation.com/en-us/support/hardware/reinstall-playstation-system-software-safe-mode) to upgrade your PS5.
