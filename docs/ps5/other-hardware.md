# Support for other hardware

| Hardware Type / Name / Model | What it takes to make it work |
| :--- | :--- |
| **Startech 10G2A1C25EPD-USB-HUB** | Plug and play. 2.5GBe Ethernet and USB ports fully utilized by Linux. |
| **DELL WD19TBS Dock Station** | Must use a **Baseus Mini OTG Adaptor (USB 3.1 to Type-C)**. Supports USB Type-A ports and 1GBe Ethernet. |
| **Orico Type-C 6-in-1 (PW11-6PR)** | Gigabit Ethernet and USB-A ports work. HDMI port remains untested. |
| **Archer TX10UB Nano (AX900)** | USB Adapter works smoothly for both Wi-Fi 6 and Bluetooth 5.3. |
