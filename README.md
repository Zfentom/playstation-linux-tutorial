# PS4 / PS5 Linux Guide

A community-made guide for installing Linux on PlayStation 4 and PlayStation 5 consoles.

All information documented here is publicly available and intended for educational and research purposes.

The site has beeb fully built in vitepress.

## Disclaimer
Installing Linux may cause data loss or instability. Proceed at your own risk.
